# longway.nvim

A Neovim plugin written in Fennel using [nfnl](https://github.com/Olical/nfnl).

## Features

- Example plugin scaffolding
- Written in Fennel for a functional programming experience
- Compiled to Lua using nfnl

## Requirements

- Neovim >= 0.9.0
- [nfnl](https://github.com/Olical/nfnl) (for development only if you commit compiled Lua)

## Installation

### Using [lazy.nvim](https://github.com/folke/lazy.nvim)

```lua
{
  "em-schmidt/longway.nvim",
  config = function()
    require("longway").setup({
      enable = true,
      debug = false,
    })
  end,
}
```

### Using [packer.nvim](https://github.com/wbthomason/packer.nvim)

```lua
use {
  "em-schmidt/longway.nvim",
  config = function()
    require("longway").setup({
      enable = true,
      debug = false,
    })
  end
}
```

## Configuration

Default configuration:

```lua
require("longway").setup({
  enable = true,
  debug = false,
})
```

## Usage

```vim
:LongwayHello   " Test the plugin
:LongwayInfo    " Show plugin info
```

Or from Lua:

```lua
require("longway").hello()
local info = require("longway").get_info()
```

## Development

This plugin is written in Fennel and compiled to Lua using nfnl.

### Setup for Development

1. Install [nfnl](https://github.com/Olical/nfnl)
2. Clone this repository
3. Edit Fennel files in `fnl/longway/`
4. nfnl will automatically compile them to `lua/longway/`

### Project Structure

```
.
├── fnl/longway/        # Fennel source files
│   ├── init.fnl       # Main module
│   ├── config.fnl     # Configuration handling
│   └── core.fnl       # Core functionality
├── lua/longway/        # Compiled Lua files (auto-generated by nfnl)
├── plugin/            # Plugin entry point
│   └── longway.lua
└── .nfnl.fnl          # nfnl configuration
```

### Compiling Fennel to Lua

If you have nfnl installed, it will automatically compile Fennel files when you save them.
Alternatively, you can manually compile with nfnl's commands.
